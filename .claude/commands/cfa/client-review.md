# Client Review

Prepare a client meeting review using the Client Review workflow from `workflow-wealth-management`.

## What It Does
Produces meeting prep with portfolio performance summary, asset allocation drift analysis, attribution (allocation/selection effects), risk metrics, market outlook, and action items.

## Agent
Routes to `cfa-quant-risk-analyst`.

## Key Tools
`risk_adjusted_returns`, `risk_metrics`, `brinson_attribution`, `factor_attribution`, `mean_variance_optimization`

## Usage
Provide client name and portfolio details for personalized review prep.
