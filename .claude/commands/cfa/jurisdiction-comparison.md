Compare offshore fund jurisdictions side-by-side for optimal domicile selection: $ARGUMENTS

Use the following analytical framework:

1. **Side-by-side comparison matrix** — Use `jurisdiction_comparison` to build a weighted multi-jurisdiction comparison across all selected jurisdictions (up to 10+)
2. **Channel Islands detail** — Use `channel_islands_fund_structure` for Jersey JPF/Expert/QIF and Guernsey PIF/QIF/RQIF specifics
3. **Singapore detail** — Use `singapore_vcc_structure` for VCC standalone/umbrella, MAS licensing, S13O/S13U/S13D tax incentives
4. **Hong Kong detail** — Use `hong_kong_fund_structure` for OFC/LPF, unified fund exemption, carried interest concession
5. **Middle East detail** — Use `middle_east_fund_structure` for DIFC/ADGM fund structures, sharia compliance, free zone economics
6. **Cayman/BVI baseline** — Use `cayman_fund_structure` for Cayman/BVI comparison baseline
7. **Luxembourg/Ireland baseline** — Use `lux_ireland_fund_structure` for EU-regulated comparison baseline

**Output a jurisdiction selection memo** with weighted ranking, optimal domicile recommendation, total cost of ownership (1/3/5/10 years), distribution passport map, and sensitivity analysis showing how the recommendation changes under different priority weights.
