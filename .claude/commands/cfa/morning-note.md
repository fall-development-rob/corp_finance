# Morning Note

Daily morning research note using the Morning Note workflow from `workflow-equity-research`.

## What It Does
Compiles market movers, earnings calendar, sector rotation signals, and key data releases into a 2-4 page morning brief.

## Agent
Routes to `cfa-equity-analyst`.

## Key Tools
`fmp_quote`, `fmp_earnings`, `fmp_sector_performance`, `fmp_gainers_losers`

## Usage
Run without arguments for today's morning note, or specify a date.
