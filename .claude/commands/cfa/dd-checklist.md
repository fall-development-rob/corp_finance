# DD Checklist

Generate a due diligence checklist using the DD Checklist workflow from `workflow-private-equity`.

## What It Does
Produces a comprehensive, categorized DD checklist covering commercial, financial, legal, operational, and management diligence. Each item has priority (critical/important/nice-to-have) and status tracking.

## Agent
Routes to `cfa-private-markets-analyst`.

## Usage
Provide the target company sector and deal type (buyout, growth, add-on) for tailored checklist.
