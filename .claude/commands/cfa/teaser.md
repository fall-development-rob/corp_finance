# Deal Teaser

Draft a deal teaser using the Teaser workflow from `workflow-investment-banking`.

## What It Does
Produces a 1-2 page anonymous or named deal teaser with investment highlights, headline metrics, and transaction overview.

## Agent
Routes to `cfa-private-markets-analyst`.

## Key Tools
`fmp_profile`, `fmp_key_metrics`

## Usage
Provide company info. Agent will ask whether teaser should be anonymous or named.
