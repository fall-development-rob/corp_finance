# Sector Overview

Sector-level analysis using the Sector Overview workflow from `workflow-equity-research`.

## What It Does
Produces a 10-15 page sector overview: market landscape, competitive positioning, growth drivers, headwinds, valuation heat map across peers.

## Agent
Routes to `cfa-equity-analyst`.

## Key Tools
`comps_analysis`, `fmp_sector_performance`, `fmp_profile`, `fmp_key_metrics`

## Usage
Provide a sector name (e.g., "semiconductors", "healthcare", "fintech").
