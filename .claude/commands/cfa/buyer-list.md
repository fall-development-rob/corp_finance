# Buyer List

Build a strategic and financial buyer list using the Buyer List workflow from `workflow-investment-banking`.

## What It Does
Identifies and ranks potential acquirers across strategic buyers (industry peers, adjacencies) and financial sponsors (PE firms with sector focus). Tiered ranking: Tier 1 (most likely) through Tier 3 (possible).

## Agent
Routes to `cfa-private-markets-analyst`.

## Key Tools
`comps_analysis`, `fmp_profile`, `fmp_stock_screener`

## Usage
Provide the target company details and sector context.
